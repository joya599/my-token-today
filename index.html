⛩️ GitHub Gist 云同步

📖 配置指南

1. 准备 GitHub 账号

前往 github.com 注册账号并登录。

2. 获取 Token (密钥)

前往 Settings -> Developer settings -> Tokens (classic)。 生成时务必勾选 gist 权限。
※ 提示：Token 仅显示一次，请务必先保存到记事本。

3. 获取 Gist ID

在 gist.github.com 创建一个文件。 保存后，地址栏链接末尾的字母数字串即为 ID。
